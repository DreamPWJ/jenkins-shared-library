### DataX离线数据同步工具/平台 异构数据源之间高效的数据同步功能

#### 安装DataX

cd /my
wget http://datax-opensource.oss-cn-hangzhou.aliyuncs.com/datax.tar.gz
tar -xzvf datax.tar.gz

cd /my/datax/bin && python datax.py /my/datax/job/datax_job.json
